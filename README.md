# 📊 Sales & Marketing Analytics Dashboard

An interactive **Power BI** dashboard that analyzes retail sales data to provide actionable business insights through data visualization, KPI tracking, customer segmentation, and profitability analysis. The project demonstrates end-to-end data transformation, DAX calculations, and dashboard development for data-driven decision-making.

---

## 🚀 Features

- Data cleaning and transformation using **Power Query**
- Interactive Power BI dashboards
- KPI cards for business performance monitoring
- Customer segmentation analysis
- Product-wise sales and profit analysis
- Country-wise sales performance
- Monthly, Quarterly, and Yearly sales tracking
- Profitability analysis
- Interactive slicers and filters for dynamic reporting

---

## 📈 Key KPIs

- Total Sales
- Total Profit
- Average Sales Value
- Average Sales per Customer
- MTD (Month-to-Date) Sales
- QTD (Quarter-to-Date) Sales
- YTD (Year-to-Date) Sales
- Customer Spending Score
